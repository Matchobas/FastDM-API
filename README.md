# Fast Dungeon Master API

An API with the goal of offering some tools to apply some Dungeons & Dragons rules faster and save some time during game sessions, this is the API that can be used to get information to create a client side in your own manner.

Some of the rules applied will be:
<ol>
  <li>Check highest passive perception of your players</li>
  <li>Check all passive perceptions higher than a given number</li>
  <li>Roll of initiative</li>
</ol>

This API is being done for a college final project, it will be free to anyone that has an interest in creating your own tool with this functionalities or to create a tool in future projects.
